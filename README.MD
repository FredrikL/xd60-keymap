# XD60 Rev 2.0
Custom keymap for the XD60 using [QMK](https://github.com/qmk/qmk_firmware).

## Features
- Backlight control
- RGB underglow control(animation, hue, saturation, etc)
- 2x 1u Shift to support common keysets
- Volume and Media control
